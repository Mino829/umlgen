# umlgen導入サポート

Javaプロジェクトへumlgenを導入し、Pull Requestごとに設計変更を確認できる状態まで設定するサービスです。

## Starter

開始価格は**1リポジトリ3,000円**です。作業を始める前に対象リポジトリと要望を確認し、最終的な金額と作業範囲を提示します。合意するまで料金は発生しません。

含まれる内容：

- umlgenのGitHub Actions workflow導入
- Javaソースパス、除外パス、関係の深さの調整
- Pull RequestでPlantUML／SVG artifactが生成されることの確認
- リポジトリREADMEへの利用方法の追記
- 導入後30日間の軽微な設定調整

納品物は原則として、利用者が確認してマージできるPull Requestです。

## 対象

- GitHubで管理しているJavaプロジェクト
- GitHub Actionsを利用できるリポジトリ
- 標準的なMaven、Gradle、または複数モジュール構成

大規模な独自ビルド、GitHub Enterprise Server、セルフホストランナーなどは、確認後に追加対応として見積もる場合があります。

## 相談から導入まで

1. [導入相談Issue](https://github.com/Mino829/umlgen/issues/new?template=onboarding.yml)を作成する
2. 公開情報だけで、構成と実現したいことを確認する
3. 作業範囲、価格、進め方に合意する
4. 利用者が許可した方法で設定を作成する
5. テスト用Pull Requestでartifactを確認して納品する

相談だけで終了しても料金はかかりません。作業開始前であれば無料でキャンセルできます。作業開始後の変更やキャンセルは、完了済みの内容を確認して個別に合意します。

## セキュリティ

導入相談Issueは公開されます。次の情報は書かないでください。

- private repositoryのソースコード
- アクセストークン、APIキー、secret
- 社外秘のリポジトリ名、組織名、URL
- 顧客情報や個人情報

private repositoryの場合は、公開可能なディレクトリ構成と利用技術だけを記載してください。ソースコードを外部の図生成サービスへ送信する構成にはしません。

## サポート範囲

30日間の設定調整には、ソースパス、除外、深度、出力形式などの軽微な変更を含みます。新しい言語対応、umlgen本体の新機能開発、他のCIサービスへの移植は別途相談となります。

このサービスは立ち上げ段階です。対応内容や価格は、実際の導入結果をもとに変更する場合があります。すでに合意した作業の条件を後から変更することはありません。
