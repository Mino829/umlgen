## 短い投稿

```text
Java/SpringのPull Requestで、クラスと依存関係の変更を自動で図にするOSS「umlgen」を開発しています。

実際のJavaチーム3組へ、GitHub Actionsの導入と4週間の設定支援を無料で行います。週1回以上PRを作り、率直なフィードバックをいただける方を募集しています。Public／Privateどちらでも応募できます。
```

## コミュニティ向け

```text
Javaプロジェクトのコードレビューについて相談です。

Pull Requestのbase/headから変更クラスと周辺の依存関係をPlantUML/SVGにするCLI「umlgen」を作っています。ソースを外部サービスへ送らず、GitHub Actions内で完結します。

現在、次に作る機能を決めるため、Javaを使う3チーム限定で4週間のデザインパートナーを募集しています。

提供:
- GitHub Actionsへの無料導入
- ソースパスや除外、図の深度の調整
- 4週間の軽微な設定支援

お願い:
- 週1回以上Pull Requestで利用
- 1週目と4週目に各20分程度のフィードバック
- 良かった点だけでなく、役に立たなかったケースの共有

終了後もOSS版はそのまま無料で使え、契約や支払いを求めることはありません。

デモと応募条件:
https://github.com/Mino829/umlgen/blob/main/docs/design-partner.md
```
